Mythic Feed

Real-time WoW reports to twitter (Currently ghetto)

View project on GitHub

Current Features:

  • Report boss downs from Highmaul.
  • Real-time reading of combat log (or as fast as the game can export it anyway).

KNOWN ISSUES and PLANNED CHANGES

  • Doesn't report difficulty.
  • WILL break if you do things it doesn't like.
  • Timestamps and fight lengths will be added.

WHAT WILL (probably) NOT BE ADDED

  • Any other sort of reports. These reports are generated completely from the combat log.

While I can use Battle.net API, it simply doesn't seem like it would fit the point of this application to be a real-time job anymore.
